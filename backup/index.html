The Four Pillars of Object-Oriented Programming

What is Object-Oriented Programming(OOP)? OOP is a programming paradigm based on the concept of "object" which composes of behaviours and information. Many of the most widely used programming languages support OOP such as Java, C#, Python and many more. There are four main principles of OOP, which are Abstraction, Encapsulation, Inheritance, and Polymorphism.
